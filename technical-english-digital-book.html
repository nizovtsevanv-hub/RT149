UNIT 1

Present Simple + a/an + plurals

Present Simple + артикли a/an + множественное число

Unit goal: Say simple facts and checks about a technical drawing.

Цель: Говорим простые факты и проверки по техническому чертежу.

You will learn: Present Simple • he/she/it + -s • do/does not • Do/Does questions • a/an • plurals

Изучаем: Present Simple • -s с he/she/it • отрицание • вопросы • a/an • множественное число

Unit 1 Vocabulary

Unit 1 Rule Cards

Rule 1: Present Simple = routine / fact / check

Use Present Simple for regular actions, facts and checks.

Используем Present Simple для обычных действий, фактов и проверок.

I read a drawing. We check a thread. The tool rotates. The engineer reads a note.

Rule 2: He / She / It + -s / -es

The engineer reads a note. The tool rotates clockwise. The part matches the size.

С he/she/it добавляем окончание -s или -es.

Rule 3: Negative: do not / does not

I do not read. We do not change the note. The part does not rotate. The washer does not fit.

Отрицание: do not (I/we/you/they) • does not (he/she/it).

Rule 4: Questions: Do / Does ...?

Do you read a drawing? Do we check a diameter? Does the tool rotate? Does the engineer check a note? Does the part meet the spec?

Вопрос начинается с Do или Does.

Rule 5: a / an

Use a before consonant sound. Use an before vowel sound.

a перед согласным звуком, an перед гласным.

a thread • a washer • a drawing • an angle • an engineer

Rule 6: Plurals

a thread → threads • a washer → washers • a box → boxes • a body → bodies

Множественное число: обычно +s; после -s, -x, -ch, -sh → +es; -y → -ies.

UNIT 2

this / that / these / those • there is / there are • have / has

This / That • There is / There are • Have / Has

Unit goal: Point to objects, describe what is on the drawing, and say what a part has.

Цель: Показываем объекты, описываем, что есть на чертеже, и говорим, что имеет деталь.

You will learn: this/that/these/those • there is/are • have/has

Изучаем: указательные слова • there is/are • have/has

Unit 2 Vocabulary

Unit 2 — Rule Cards

Rule 1: this / that / these / those

Use this and these for near objects. Use that and those for far objects.

This/these — близко. That/those — далеко.

This part is clean. • That note is important. • These washers are ready. • Those holes are small.

Rule 2: There is / There are

Use there is for one object. Use there are for two or more objects.

There is — один объект. There are — два или больше.

There is a thread on the part. • There is a note on page 3. • There are two washers in the box. • There are three holes in the panel.

Rule 3: have / has

Use have with I / we / they. Use has with he / she / it.

Have — с I / we / they. Has — с he / she / it.

I have a drawing. • We have a clean part. • The tool has a handle. • The stud-bolt has a thread.

Unit 3 Vocabulary

Unit 3 — Rule Cards

Rule 1: Imperative = command / instruction

Use the verb without to and without a subject.

Команда строится просто: глагол без to и без подлежащего.

Insert a cap into a collet. • Tighten the handle. • Rotate the tool clockwise. • Remove the part carefully.

Rule 2: First / Then / Finally

Use these words to order technical steps.

Используем эти слова, чтобы выстроить шаги процесса.

First, insert a cap. • Then, tighten the handle. • Finally, remove the tool.

Rule 3: Prepositions

Use: in • on • into • under • between

in a box • on a surface • into a hole • under a washer • between two parts

Предлоги показывают положение детали.

Rule 4: can / can't

Can shows possibility. Can't shows that an action is not possible.

Can — можно / может. Can't — нельзя / не может.

The tool can rotate. • The part can fit the hole. • The washer can't hold the cap alone. • We can verify the size.
